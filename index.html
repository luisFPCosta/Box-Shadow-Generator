What is a Box-Shadow?

The box-shadow CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas. A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color.

What does the box-shadow rule do?

Based on some parameters, it adds a shadow to the elements, which can generate different effects, such as: highlighting or levitation. Many projects implement this rule, meaning that the developer has to resort to its documentation whenever he needs to do something. The box-shadow parameters are:

  • offset-x;
  • offset-y;
  • color;
  • opacity;
  • blur intensity;
  • spread
  • inset

Let's see what each of them does next, but note the complexity of rule configuration.

The value of offset-x in box-shadow:

The offset on the x-axis represents the displacement of the shadow generated on the horizontal axis, which can be negative or positive

The value of offset-y in box-shadow:

The offset on the y-axis represents the displacement of the shadow generated on the vertical axis, which can be negative or positive

The color value in box-shadow:

The color changes the color of the generated shadow, we can inform in various formats, such as: Hexadecimal or RGB RGB ends up being more used because we can control the opacity, changing the alpha value (RGBA)

The opacity value in box-shadow:

The opacity is changed with color, using the RGBA standard, the accepted value is from 0 to 1 Since 0 is completely invisible, we can control the intensity with .5 for example, which represents 50% visibility, to reach intermediate values

The blur value in box-shadow:

With blur we can make the shadow more 'smoky' and not solid, which is its standard The value is placed in px, the larger the more transparent the shadow also ends up being, as it also gains more area, giving a blurred aspect to the shadow effect

The spread value in box-shadow:

By changing the spread the shadow will spread more along the page, increasing its size The value is also entered in pixels, allowing for negative values

The value of inset in box-shadow:

The inset can be inserted into the rule and leaves the shadow inside the element That is, instead of the shadow being external, it becomes internal
